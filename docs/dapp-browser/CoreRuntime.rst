===========
CoreRuntime
===========

The CoreRuntime is exported from the `bcc bundle </bcc/bcc-bundle>`_ instance. Is available after `initializeEvanNetworkStructure </dapp-browser/functions.html#initializeevannetworkstructure>`_ is executed.
