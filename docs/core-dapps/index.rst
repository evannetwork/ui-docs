General
=======
The ui-core-dapps are several projects that provides basic functionalities to interact with the blockchain using the evan-network core libraries.
For a detailed overview, how this dapps are working, have a look at the following page: `DApps overview <https://evannetwork.github.io/dapps/dapps>`_.

This category describes the DApps that provides exported components and services, that can be used within your DApps.
